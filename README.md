# Zakovat
A Telegram bot that shares Zakovat questions

# How does this bot work?

The architecture of this bot is really simple.

In Zakovat.py there's a function that requests to questions api of https://www.zakovatklubi.uz/ then it stores to variables then returns

And in bot.py, it's really obvious it handles messages from users and respondes as commends
